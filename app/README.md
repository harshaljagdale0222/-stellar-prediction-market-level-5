# 🌌 StellarPredict - Decentralized Prediction Market

StellarPredict is a high-performance, decentralized prediction market built on the **Stellar Network**. It allows users to trade on the outcome of real-world events with **zero gas fees** and near-instant finality, leveraging Stellar's advanced blockchain technology.

## 🚀 Live Demo & Video
- **Live Application:** [StellarPredict on Vercel](https://stellar-prediction-market-level-5.vercel.app/)
- **Demo Walkthrough:** [Watch the Demo on Loom](https://www.loom.com/share/5b7d09098683445f942009861962b5bc)

---

## ✨ Key Features
- **Gasless Transactions:** Powered by Stellar fee sponsorship, allowing users to trade without worrying about network fees.
- **Real-time Settlement:** Fast block times enable quick trade executions and result settlements.
- **Secure Wallet Integration:** Seamless connection with **Freighter Wallet** for secure transaction signing.
- **Dynamic Odds:** Automated Market Maker (AMM) logic ensures fair and transparent pricing.
- **User Dashboard:** Track your positions and performance in real-time.

-   **Blockchain:** Stellar Network (Testnet)

## 🛠️ Getting Started

### Prerequisites
- Node.js 18+ 
- [Freighter Wallet Extension](https://www.freighter.app/)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/harshaljagdale0222/-stellar-prediction-market-level-5.git
   cd -stellar-prediction-market-level-5
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Access the app at `http://localhost:3000`
